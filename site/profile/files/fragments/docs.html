FAQ

Where can I find instructions for using the classroom tools? (PF, PP)

We use a tool that allows each student to perform classroom exercises by writing code in a graphical editor and enforcing it on a virtualized agent node. Read the tutorial.

Where can I find tutorials on using the Git server? (GSWP, PF, PP)

We run a self-hosted Git server in the classroom. Depending on the class, you will use either Gitea or GitLab. In both cases, the experience is very close to that of GitHub. We suggest that if you want to practice your Git skills before class that you sign up for a GitHub account and then try out this tutorial.

Do I need to install a PDF reader? (GSWP, PF, PP)

Probably not. Most modern browsers have a PDF reader built-in or use a platform standard reader. If that is not the case for you, then readers such as Adobe Acrobat Reader can be used free of charge.

How can I correct issues with the video conferencing (Zoom.us) system? (GSWP, PF, PP)

Check the Zoom.us network requirements. Their support portal might also have other suggestions.

How do I accept a self-signed certificate? (PF, PP)

You can find official instructions in the Puppet Enterprise documentation.

What if I can't accept a self-signed certificate? (PF, PP)

Please let us know in the feedback form at the end of the tests. We may be able to help you.
